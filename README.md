# Home Assistant Tesla App Key Hosting
HomeAssistant Integration for Hosting Tesla Integration Key. This integration is intended for users who use Nabu Casa.

## Installation

1. Install HACS - [Instructions](https://www.hacs.xyz/docs/use/download/download/)
2. Add the custom repo to your HACS Store - [Instructions](https://www.hacs.xyz/docs/faq/custom_repositories/)
3. Add the integration to Home Assistant by searching for "Tesla App Key Host" in the HACS Store
4. Configure the integraiton by going to Settings -> Devices and Services -> Add integration
5. Enter your key
6. Enjoy
